## 1. 실행 방법
지킬 페이지를 실행하는 방법은 크게 두 가지 방법이 있다.
### 1.1. 로컬
1. 루비 설치
2. 루비 터미널에서 다음 명령어 입력
  ```shell
  bundle exec jekyll serve
  ```
3. 브라우저에 다음 주소 입력
  ```url
  http://127.0.0.1:4000/{프로젝트 이름}
  ```
